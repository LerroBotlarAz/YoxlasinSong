## :toolbox: Getting Started

### :gear: Installation

```bash
git clone https://github.com/xGuliyev/MusicBot
cd MusicBot
pip3 install -U pip
pip3 install -r requirements.txt
nano .env
cp sample.env .env
python3 -m bot
```
